# Очистка базы данных и создание IT-администратора

## Способ 1: Использование PowerShell скрипта (Рекомендуется)

### Шаги:

1. Откройте PowerShell в корневой папке проекта
2. Выполните команду:
   ```powershell
   .\scripts\reset-db.ps1
   ```
3. Подтвердите действие, введя `YES`
4. Скрипт автоматически:
   - Удалит все данные из БД
   - Создаст IT-администратора
   - Выведет данные для входа

### Данные для входа:
- **Email:** admin@tvcompany.com
- **Пароль:** admin123

---

## Способ 2: Вручную через psql

### Шаги:

1. Откройте терминал
2. Подключитесь к базе данных:
   ```bash
   psql -h localhost -U postgres -d tvcompany
   ```
3. Выполните SQL скрипт:
   ```sql
   \i docs/db/reset-and-create-admin.sql
   ```
4. Или скопируйте и выполните SQL команды из файла `docs/db/reset-and-create-admin.sql`

---

## Способ 3: Через pgAdmin или другой GUI клиент

1. Откройте pgAdmin
2. Подключитесь к вашей базе данных `tvcompany`
3. Откройте файл `docs/db/reset-and-create-admin.sql`
4. Выполните скрипт через Query Tool

## ВНИМАНИЕ! ⚠️

**Это действие удалит ВСЕ данные из базы данных без возможности восстановления!**

Удаляются данные из следующих таблиц:
- users (все пользователи)
- applications (все заявки)
- contracts (все договоры)
- notifications (все уведомления)
- documents (все документы)
- shows (все передачи)
- messages (все сообщения чата)
- и другие...

## После очистки

1. Перейдите на http://localhost:3000/auth
2. Войдите с данными:
   - Email: `admin@tvcompany.com`
   - Пароль: `admin123`
3. Вы будете перенаправлены в панель IT-администратора
4. Создайте новых пользователей через раздел "Управление сотрудниками"

## Изменение пароля администратора

Если хотите использовать другой пароль, сгенерируйте bcrypt хэш:

```javascript
const bcrypt = require('bcryptjs');
const password = 'ваш_новый_пароль';
const hash = bcrypt.hashSync(password, 10);
console.log(hash);
```

Замените значение `password_hash` в SQL скрипте на полученный хэш.
