# Director Dashboard Documentation

## Overview
The Director Dashboard provides comprehensive visibility into company-wide performance metrics, staff KPIs, and financial statistics. This is the highest-level administrative interface with full access to all business intelligence.

## Features

### 1. **Gradient Header**
- Modern gradient background (Primary Indigo 600 → 500 → 600)
- Quick action buttons:
  - **Export**: Download reports (placeholder for future implementation)
  - **Refresh**: Reload all statistics

### 2. **Advanced Filters**
- **Date Range**: Filter statistics by start and end dates
- **Role Filter**: View stats for specific roles (agents, commercial department, or all)
- Modern input styling with focus states and Primary Indigo accents

### 3. **Key Metrics Cards**
Four prominent statistics cards with hover effects:

1. **Total Revenue** (Gradient card)
   - Shows total revenue for selected period
   - Premium gradient background
   - Dollar sign icon

2. **Approved Applications**
   - Number of approved applications
   - Shows percentage (approved/total)
   - Check circle icon
   - Hover animations

3. **Active Clients**
   - Count of unique active clients
   - Users icon
   - Hover border effects

4. **Average Deal Size**
   - Average revenue per application
   - Trending up icon
   - Responsive hover states

### 4. **Interactive Charts**
Three professional Chart.js visualizations:

#### Revenue by Staff
- Bar chart showing revenue generated by each employee
- Primary Indigo color scheme
- Rounded corners on bars
- Staff names as labels

#### Application Distribution
- Doughnut chart with three segments:
  - **Approved**: Primary color (solid)
  - **Rejected**: Neutral gray
  - **In Progress**: Primary color (lighter)
- Legend positioned at bottom

#### Staff KPI Chart
- Bar chart showing approval percentage for each staff member
- Helps identify top performers
- Primary Indigo gradient bars
- Full-width display

### 5. **Staff KPI Table**
Detailed performance table with:

**Columns:**
- **Employee**: Name and email
- **Role**: Role badge with Primary styling
- **Applications**: Total count
- **Approved**: Count with Primary color emphasis
- **KPI**: Color-coded badge (90%+ = Primary-100, 80-89% = Primary-50, <80% = Neutral)
- **Revenue**: Total revenue generated
- **Commission**: Earned commission
- **Shows**: Number of unique shows handled (with TV icon)

**Features:**
- Gradient table header
- Row hover effects (Primary-50 background)
- Empty state with icon when no data
- Responsive design

### 6. **Company Summary Cards**
Two summary cards with gradient backgrounds:

#### General Statistics Card
- Total applications count
- Approved applications
- Approval rate percentage
- Unique shows count
- Icons for each metric

#### Financial Metrics Card
- Total revenue
- Average deal size
- Active clients
- Revenue per client (calculated)
- Dollar sign and chart icons

## Technical Details

### Data Loading
```typescript
GET /api/director/staff-kpi
Parameters:
  - start_date (optional): Filter start date
  - end_date (optional): Filter end date
  - role_filter (optional): 'all', 'agent', or 'commercial'

Response:
{
  staff_kpi: StaffKPI[],
  company_stats: CompanyStats
}
```

### Interfaces

```typescript
interface StaffKPI {
  id: string
  name: string
  email: string
  role: string
  role_display: string
  total_applications: number
  approved_applications: number
  rejected_applications: number
  revenue: number
  kpi: number
  commission: number
  unique_shows: number
  shows: string[]
}

interface CompanyStats {
  total_revenue: number
  total_applications: number
  approved_applications: number
  active_clients: number
  unique_shows: number
  average_deal_size: number
  approval_rate: number
}
```

### Chart Configuration
- **Library**: Chart.js with react-chartjs-2
- **Color Palette**: Primary Indigo (rgb(99, 102, 241))
- **Chart Options**: 
  - Responsive: true
  - Legends: Hidden on bar charts, bottom on doughnut
  - Grid: Primary color with low opacity
  - Border radius: 8px for bars

## Design System

### Colors
- **Primary Gradient**: `from-primary-600 via-primary-500 to-primary-600`
- **Card Backgrounds**: White with subtle borders
- **Hover States**: `hover:shadow-lg`, `hover:scale-105`
- **KPI Badges**:
  - High (≥90%): `bg-primary-100 text-primary-700`
  - Medium (80-89%): `bg-primary-50 text-primary-600`
  - Low (<80%): `bg-neutral-200 text-neutral-700`

### Icons (Lucide React)
- `BarChart3`: Dashboard header and statistics card
- `Filter`: Filters section
- `Calendar`: Apply filters button
- `Download`: Export button
- `RefreshCw`: Refresh button (animates on loading)
- `DollarSign`: Revenue metrics
- `CheckCircle`: Approved applications
- `Users`: Client metrics
- `TrendingUp`: Growth indicators
- `UserCheck`: Staff table
- `FileText`: Empty state
- `Tv`: Show count
- `PieChart`: Distribution chart header

### Spacing & Layout
- Container padding: `p-6` to `p-8`
- Card gap: `gap-6`
- Border radius: `rounded-xl` (12px)
- Shadow: `shadow-sm` to `shadow-lg`

## User Experience

### Loading State
- Centered spinner with Primary color
- Displayed while fetching data
- Non-blocking for subsequent interactions

### Empty State
- Displayed in table when no data matches filters
- Friendly icon and message
- Encourages filter adjustment

### Hover Effects
- Cards scale slightly (105%)
- Shadows intensify
- Border colors change to Primary
- Table rows highlight with Primary-50

### Responsive Design
- Grid layouts adjust for mobile/tablet/desktop
- Tables scroll horizontally on small screens
- Charts maintain aspect ratio
- Filters stack vertically on mobile

## Access Control
- **Required Role**: `director`
- **Authentication**: JWT token required
- **API Authorization**: Server validates director role
- **Data Scope**: Full company visibility (no user-based filtering)

## Performance Considerations
- Charts render client-side using Canvas
- Data cached in component state
- Manual refresh required for updates
- Filters trigger new API requests
- Table pagination not implemented (assumes reasonable data size)
