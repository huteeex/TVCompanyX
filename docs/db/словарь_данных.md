## users

| Столбец        | Тип             | Ограничения                        | Описание                               |
|----------------|-----------------|------------------------------------|----------------------------------------|
| id             | uuid            | PK, DEFAULT uuid_generate_v4()     | Уникальный идентификатор пользователя  |
| first_name     | varchar(100)    | NOT NULL                           | Имя                                    |
| last_name      | varchar(100)    | NOT NULL                           | Фамилия                                |
| middle_name    | varchar(100)    | —                                  | Отчество                               |
| email          | citext          | NOT NULL, UNIQUE                   | Электронная почта                      |
| password_hash  | text            | —                                  | bcrypt-хеш пароля                      |
| role           | user_role       | NOT NULL                           | Роль: customer / agent / commercial / accountant / admin / director |
| is_active      | boolean         | NOT NULL, DEFAULT true             | Аккаунт активен                        |
| phone          | varchar(30)     | —                                  | Контактный телефон                     |
| created_at     | timestamptz     | NOT NULL, DEFAULT now()            | Дата регистрации                       |
| updated_at     | timestamptz     | NOT NULL, DEFAULT now()            | Дата обновления                        |

## audit_log

| Столбец      | Тип            | Ограничения                       | Описание                                                       |
|--------------|----------------|-----------------------------------|----------------------------------------------------------------|
| id           | bigserial      | PK, AUTOINCREMENT                 | Порядковый номер записи                                        |
| entity       | varchar(100)   | NOT NULL                          | Название таблицы: applications, users и др.                    |
| entity_id    | uuid           | NOT NULL                          | ID изменённой записи                                           |
| action       | varchar(20)    | NOT NULL                          | Действие: INSERT / UPDATE / DELETE / STATUS_CHANGE             |
| changed_by   | uuid           | FK → users.id                     | Автор изменения (NULL = система/триггер)                       |
| changed_at   | timestamptz    | NOT NULL, DEFAULT now()           | Дата и время действия                                          |
| payload      | jsonb          | —                                 | Дельта изменений: {"old": {...}, "new": {...}}                  |
